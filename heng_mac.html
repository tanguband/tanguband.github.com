変愚蛮怒 は 公式の開発メンバー と Mac版 開発者 backwardsEric 氏の協力で日々アップデートされています

　Mac版の配布許可を頂いたのでこちらからダウンロードが可能です

　 変愚蛮怒.app Ver 3.0.0 Alpha 79 （ 対応 OS 10.15 ~ 13.x ）

　　1.　上のリンクをクリックしてダウンロードします
　　2.　展開された変愚蛮怒アプリケーションを好きなフォルダに移動します
　　3.　右クリックから '開く' を選択してゲーム開始　( 次からはダブルクリックで開きます )

　　ダウンロードはアプリケーションのみ ( ソースは Hengband for macOS )
　　公式のソース内容には手を加えずに Mac 用にビルドしたものです（ スコア送信可能 ）
　　ゲーム起動後に以下の場所にファイルが自動的に作成されます
　　　セーブファイル他　~/Documents/Hengband-3.0.0
　　　ウインドウの設定　~/Library/Preferences/jp.osdn.hengband.plist

　注意点
　　ユニバーサルバイナリ（ Apple Silicon & intel CPU 対応 ）
　　アルファ版のため問題が起きることがありますが次のバージョンで改善することが多いです
　　バージョンアップ後もセーブデータは引き継ぎ可能ですので最新版の使用を推奨します
　　フォントの種類により文字幅が広がることがあります ( osaka等幅 など )
　　JISキーボードのバックスラッシュの入力は " オプション + ¥ ( 円マーク ) "
　　（フォントの種類により 円マーク"¥" が出力されますがゲーム内では "\" で認識、マクロで使用可）
　　未サポートの機能（ 日本語入力・テキストのコピー＆ペースト・保存ディレクトリの変更 ）

　効果音とBGM
　　メニューバー ＞ 設定 ＞ 効果音とBGM… を開くと効果音とBGMのオンオフや音量調整が可能です

　　自分の好みの効果音やBGMを使用したい場合は
　　　変愚蛮怒.app を右クリックして パッケージの内容を表示 を選択
　　　効果音は Contents >> Resources >> lib >> xtra >> sound >> sound.cfg を開く
　　　BGMは Contents >> Resources >> lib >> xtra >> music >> music.cfg を開く
　　　それぞれの .cfg ファイルの内容を読んで設定してください
