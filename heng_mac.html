Mac版 変愚蛮怒 2.2.1.7 ダウンロード　（動作環境は Mac OS X 10.7 ~ 10.15）
( 最終更新　20/02/24 )

ついに変愚蛮怒のMac版が公式に対応 ( 64bitバージョン )

今後はこちらのサイトからダウンロード可能です

19/12/31のバージョンはテンキーが使えないので
暫定的にこちらで最新版をコンパイルして配布いたします。

ダウンロードはアプリのみとなっていますので、ソースが必要な方はこちらへどうぞ。
ゲームに関わる改変は一切行っていないので、ダンプの送信機能もそのままになっています。
セーブファイル・ハイスコア等が破損する可能性もありますので、ご使用は自己責任でお願いします。

コンパイル済アプリケーションについて
　~/Documents/Hengband 以下に保存ディレクトリを勝手に作ります。（変更不可）
　~/Library/Preferences/jp.sourceforge.hengband.Hengband.plist
　にGUI関連の初期設定が保存されます。
　変愚蛮怒の lib/ は Hengband.app/Contents/Resources 以下にあります。
　Finder で右クリック→パッケージの内容を表示して下さい。

未サポートの機能
・IMEによる日本語入力（ペーストは可能）
・テキストのコピー
・保存ディレクトリの変更

既知の問題
・等幅フォントの種類により文字幅が広がる ( ヒラギノなどのフォントは普通に使えます )
・JISキーボードのバックスラッシュの入力は " オプション + ¥ (円マーク) "
　（フォントの種類により 円マーク"¥" が出力されますがゲーム内では "\" で認識、マクロで使用可）

ビルド・動作の確認は macos Mojave 10.14.6 （ MacOSX10.15.sdk コマンドラインツール） で行なっています。

非公式ビルドの為、Mac版に関する連絡は gmail,twitter アカウント tanguband までお願いします。
