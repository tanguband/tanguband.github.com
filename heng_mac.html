Mac 版　変愚蛮怒

変愚蛮怒 の Mac 版は 公式の開発メンバー と backwardsEric 氏の協力で日々アップデートされています
アプリ配布の許可を頂いたのでこちらからダウンロードが可能です

Mac 版　変愚蛮怒　Ver3.0.0 Alpha 18　ダウンロード（ 対応 OS 10.14 ~ 11.2 ）

　1. 上のリンクをクリックしてダウンロードします
　2. 展開された変愚蛮怒アプリケーションを好きなフォルダに移動します
　3. 右クリックから '開く' を選択してゲーム開始　( 次からはダブルクリックで開きます )

　ダウンロードはアプリケーションのみ 　( Mac 版のソースは Hengband for macOS )

　　ゲーム起動後に以下の場所にファイルが自動的に作成されます
　　　セーブファイル他　~/Documents/Hengband-3.0.0
　　　ウインドウの設定　~/Library/Preferences/jp.osdn.hengband.plist

　公式のソース内容に手は加えていないのでスコア送信も可能です


注意点
　Alpha版のため問題が起きることがありますが次のバージョンで改善することが多いです
　バージョンアップ後もセーブデータは引き継ぎ可能ですので最新版の使用を推奨します
　フォントの種類により文字幅が広がることがあります
　JISキーボードのバックスラッシュの入力は " オプション + ¥ (円マーク) "
　（フォントの種類により 円マーク"¥" が出力されますがゲーム内では "\" で認識、マクロで使用可）
　未サポートの機能
　（日本語入力・テキストのコピー＆ペースト・保存ディレクトリの変更）
